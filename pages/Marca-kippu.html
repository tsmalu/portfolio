Projeto feito para Kippu • 2021

Kippu


Kippu é uma plataforma para gestão do transporte para funcionários, com ela é possível administrar colaboradores o uso de vale-transporte e transporte eventual.

A marca foi desenhada buscando expressar valores de praticidade e confiança, com base na ideia de intermodalidade de transporte e suas conexões. Para ativação da marca, foram projetados logotipo, identidade visual e desenvolvimento de produto, com a primeira versão do site e plataforma para web e mobile.
