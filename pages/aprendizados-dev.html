Sempre gostei de ver o design ganhando vida. Aprender HTML, CSS e JavaScript tem sido entender como o design conversa com o código. Aqui é um espaço pra aprender criando e compartilhar meus primeiros estudos e experimentos no frontend.

Esse site é o resultado dos meus primeiros passos no Frontend. Comecei com a curiosidade de entender como o design é aplicado no código e acabei transformando meu portfólio em um laboratório. Aqui registro o que aprendi em HTML, CSS e JavaScript.



Estrutura do Head e Organização do CSS

Aprendi a estruturar o <head> com metadados, fontes, favicon e tags de compartilhamento. Também organizei o CSS por função: base, layout, componentes e seções pra facilitar manutenção e garantir consistência visual. Usei versionamento simples com ?v=1 pra evitar cache antigo e integrei scripts de análise como Google Tag e Hotjar.
